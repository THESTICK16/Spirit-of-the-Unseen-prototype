Name:
  Kobolds (Dog Soldier Rework)
Version:
  0.5
Description:
  yolkati's Dog Soldier reworked as kobold sprites drop-in replacements
  for Stendhal.
Sprite details:
  - Dimensions: 24x32, 48x64, & 96x128
  - Orientation: orthogonal (N/E/S/W)
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
  - Three sprite types:
    - A short rework that is very similar to yolkati's original.
    - A taller rework similar to yokati's original.
    - A rework using a modified Cabbit base sprite for the body.
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
